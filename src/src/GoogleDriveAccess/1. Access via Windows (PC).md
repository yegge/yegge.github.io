# Google Drive Access
  
# 1. Access via Windows (PC)  
There are two ways to access a shared drive on Windows. One is through a browser, and the other is by adding it as a permanent folder on the computer.  
## Method A: Using the Web Browser (No installation needed)  
1. Open any web browser and go to ++[drive.google.com](https://drive.google.com/)++.  
2. Sign in with the Google account that has access to the drive.  
3. On the left-hand menu, click **Shared drives**.  
4. Double-click the drive named **"She Never Text Back"** to open it.  
## Method B: Installing Google Drive for Desktop (Recommended)  
This makes the shared drive appear like a USB drive or a hard drive folder on the computer.  
1. **Download:** Go to the ++[Google Drive Download page](https://www.google.com/drive/download/)++ and click **Download Drive for desktop**.  
2. **Install:** Open the downloaded file (GoogleDriveSetup.exe) and follow the installation prompts.  
3. **Sign In:** Once installed, a sign-in window will appear. Log in with the appropriate Google account.  
4. **Accessing the Files:** * Open **File Explorer** (the folder icon on your taskbar).  
    * Click on **This PC** in the left sidebar.  
    * Open the **Google Drive (G:)** drive.  
    * Open the folder labeled **Shared drives**, then open **"She Never Text Back"**.  
  
  
# 2. Access via Android (Phone or Tablet)  
## Installation  
1. Open the **Google Play Store**.  
2. Search for **"Google Drive"**.  
3. Tap **Install** (or **Update** if it’s already on the phone).  
## How to find the Shared Drive  
1. Open the **Drive app**.  
2. Tap the **Files** icon at the bottom right (it looks like a small file folder).  
3. At the top of the screen, you will see two tabs: "My Drive" and **"Shared drives"**. Tap **Shared drives**.  
4. Tap on **"She Never Text Back"** to see the files.  
  
  
# 3. Important Tips for the User  
* **Missing the Drive?** If the user follows these steps but doesn't see the "She Never Text Back" drive, they likely haven't been added as a member yet. They will need to ask the manager of that drive to invite their email address.  
* **Offline Access:** If they are using a laptop and want to work without internet, they can right-click the folder in File Explorer (Method B) and select **Offline access** > **Available offline**.  
* **Search Filter:** If they are looking for something specific, remind them to clear any active filters (like the "PDFs" or "is:encrypted" filters seen in your current view) to ensure all file types are visible.
